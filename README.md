# 📊 Análise do Brasileirão Série A 2024 com Excel

## 📖 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de praticar análise de dados utilizando o Microsoft Excel, explorando informações reais do Campeonato Brasileiro Série A 2024.

Os dados foram obtidos através de um dataset público contendo resultados das partidas, gols marcados, gols sofridos e resultados dos jogos.

O projeto busca responder perguntas relevantes sobre o desempenho das equipes ao longo da competição através de tabelas dinâmicas, gráficos e dashboard interativo.

---

## 🎯 Objetivos

- Identificar os times com mais gols marcados em casa.
- Comparar gols marcados e sofridos pelos mandantes.
- Analisar o desempenho dos clubes em termos de vitórias, empates e derrotas.
- Desenvolver habilidades em análise de dados utilizando Excel.

## 📊 Análises Realizadas

### 1. Gols Marcados em Casa

Análise da quantidade de gols marcados pelos clubes quando atuaram como mandantes.

### 2. Gols Marcados x Gols Sofridos

Comparação entre a capacidade ofensiva e defensiva dos times jogando em casa.

### 3. Resultados dos Jogos

Quantidade de:

- Vitórias
- Empates
- Derrotas

para cada equipe durante a temporada.

---

## 🔍 Principais Insights

- O Corinthians apresentou o maior número de gols marcados em casa na temporada analisada.
- Palmeiras e Fortaleza também figuraram entre os melhores ataques como mandantes.
- Algumas equipes demonstraram forte desempenho ofensivo, mas sofreram muitos gols defensivamente.
- A análise de resultados permitiu identificar os clubes mais consistentes em seus jogos como mandantes.
